# Bem-vindo ao Fichas que Caem! 🎲🌈

Obrigado por escolher o "Fichas que Caem" para o seu evento!

## Por Onde Começar?

### Se Você Quer Apenas Usar

1. Leia: **INSTALACAO.md** (15 minutos)
2. Configure o Supabase (10 minutos)
3. Teste localmente (5 minutos)
4. Faça deploy (10 minutos)
5. Use no seu evento!

### Se Você Quer Entender Melhor

1. Leia: **README.md** - Documentação completa
2. Leia: **GUIA_DE_USO.md** - Como facilitar durante eventos
3. Explore o código com: **ESTRUTURA_ARQUIVOS.md**

### Se Você Quer Personalizar

1. Leia: **CUSTOMIZACAO.md** - Guia de personalização
2. Teste suas mudanças localmente
3. Faça deploy das customizações

## Valores do Projeto

Este projeto foi desenvolvido com foco em:

🌈 **Diversidade** - Todas as vozes são importantes
🤝 **Equidade** - Acesso igual para todos
💜 **Inclusão** - Espaço seguro e acolhedor
♿ **Acessibilidade** - Funcional para todas as pessoas
🎨 **Experiência** - Interface intuitiva e agradável
📚 **Educação** - Código claro e documentado

## Filosofia

> "Não há vozes mais importantes que outras. 
> Todas as perspectivas enriquecem nossa jornada coletiva."

Este aplicativo foi criado para:
- Dar voz a todos, sem distinção
- Criar espaços seguros de expressão
- Valorizar a diversidade de pensamentos
- Facilitar escuta ativa e inclusiva
- Demonstrar visualmente a pluralidade
