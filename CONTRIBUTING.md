# Contribuindo com o Fichas que Caem 🤝

Obrigado pelo interesse em contribuir! Este projeto foi criado para promover diversidade, equidade e inclusão, e toda ajuda é muito bem-vinda.

## Como Contribuir

### Reportando Bugs

Se encontrar um bug:

1. Verifique se já não existe uma issue sobre o problema
2. Abra uma nova issue com:
   - Descrição clara do problema
   - Passos para reproduzir
   - Comportamento esperado vs. atual
   - Screenshots se aplicável
   - Informações do ambiente (navegador, OS, etc.)

### Sugerindo Melhorias

Para sugerir novas funcionalidades:

1. Abra uma issue descrevendo:
   - O problema que a feature resolve
   - Como você imagina a solução
   - Exemplos de uso
   - Impacto esperado

### Pull Requests

1. **Fork o projeto**
2. **Crie uma branch**: `git checkout -b feature/minha-feature`
3. **Faça suas mudanças**
4. **Teste localmente**: `npm run dev` e `npm run build`
5. **Commit**: `git commit -m "feat: adicionar minha feature"`
6. **Push**: `git push origin feature/minha-feature`
7. **Abra um Pull Request**

### Convenções de Código

- Use TypeScript strict mode
- Siga o padrão ESLint do projeto
- Escreva código acessível (ARIA, semântica)
- Mantenha compatibilidade mobile
- Comente código complexo
- Use nomes descritivos para variáveis

### Convenções de Commit

Usamos conventional commits:

- `feat:` nova funcionalidade
- `fix:` correção de bug
- `docs:` documentação
- `style:` formatação (não afeta código)
- `refactor:` refatoração
- `test:` testes
- `chore:` manutenção

Exemplo: `feat: adicionar filtro de palavrões`

## Áreas que Precisam de Ajuda

- Tradução para outros idiomas
- Testes automatizados
- Melhorias de acessibilidade
- Otimizações de performance
- Documentação
- Exemplos de uso
- Temas e customizações

## Código de Conduta

Este é um projeto focado em inclusão. Esperamos:

- Respeito a todas as pessoas
- Linguagem inclusiva e acolhedora
- Feedback construtivo
- Empatia e colaboração
- Foco no aprendizado coletivo

Não toleramos:

- Discriminação de qualquer tipo
- Assédio ou intimidação
- Linguagem ofensiva
- Comportamento não profissional

Abra uma issue com a tag `question` ou entre em contato.
