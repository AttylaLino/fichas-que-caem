## Como Usar

### Setup (30-45 min)
1. Clone/download o projeto
2. `npm install`
3. Configure Supabase
4. Crie `.env.local`
5. `npm run dev`
6. Teste tudo

### Deploy (10-15 min)
1. Criar repo GitHub
2. Ajustar `vite.config.ts`
3. Configurar Secrets
4. `git push`
5. Verificar deploy

### Uso em Evento
1. Compartilhar links
2. Coletar feedbacks
3. Projetar nuvens
4. Facilitar discussões
5. Exportar dados
